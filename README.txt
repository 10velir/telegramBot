Для запуска нужно:
1) Создать Базу данных city_guide
2) установить пароль от БД в файле application.properties.
3) запустить .sql скрипт по пути com.telegrambot.webservice.dao.script
    для создания данных и таблиц.
4) запустить main в классе WebserviceApplication.
5) написать телеграм боту с ником @Cities_guide_bot
    токен бота: "748297697:AAHYgrfJnD6rFwtMr2zmZhF3AGLl8Orcs_M"
6) олицетворять :)
